# Video Convertor

Batch video compressor with GPU acceleration, resource monitoring, and auto-updates.

## Features
- H.264 and H.265 encoding (GPU-accelerated via NVENC)
- Smart quality: resolution-aware CRF, source bitrate cap, auto-crop black bars
- Parallel GPU + CPU workers with dynamic resource monitoring
- Audio passthrough with AAC fallback, chapter marker preservation
- Subtitle handling and automatic deinterlacing
- Dark theme Electron UI with persistent settings
- Auto-updater via GitHub Releases

## Install
Download the latest `Video-Convertor-Setup-*.exe` from [Releases](https://github.com/numarulunu/claude-convertor/releases).

## Requirements
- FFmpeg must be installed and in PATH
- Windows 10/11
